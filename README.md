# KrishiSafar Frontend

This is a stub Vite + React 18 project using Tailwind CSS and React Router v6.

## Getting Started

1. Install dependencies:  
   `npm install`

2. Start dev server:  
   `npm run dev`

## Stack

- Vite
- React 18
- Tailwind CSS
- React Router v6

## Structure

See file tree for structure. All components and pages are currently placeholders.
